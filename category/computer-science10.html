Hands on the Python module subprocess

Oct 26, 2019

General Tips¶

  1. The method subprocess.run is preferred over the older high-level APIs (subprocess.call, subprocess.check_call and subprocess.check_output). The method subprocess.Popen (which powers the high-level APIs) can be used if you need advanced control.

  2. Avoid using system shell (shell=True) for security reasons.

  3. There are at least 2 advantages of passing a shell command as a list. First, avoid shell injection attack. Second, there is no need to you to manually escape special characters in the command.
