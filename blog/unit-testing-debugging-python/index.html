Flake8 vs MyPy vs pylint

  1. Flake8 focus on logical errors rather than stylistic errors. It strives to reduce false positives.

  2. mypy does the best on type hint.

  3. pylint performs deeper analysis and thus is slower.

To sum up, I'd try the tools in the following order Flake8 > mypy > pylint.
